🍽️ Project Title / Headline
Zomato Insights Dashboard | Interactive Business Intelligence Dashboard using Power BI

📌 Description

The Zomato Insights Dashboard is an interactive Power BI project designed to analyze restaurant performance, customer satisfaction, delivery operations, and cuisine trends. The dashboard provides valuable business insights through dynamic KPIs, charts, and slicers, helping stakeholders monitor restaurant performance, customer ratings, delivery efficiency, and order trends for better decision-making.
